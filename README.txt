Welcome to the Koblo open source repository 
Here you find the  C++ code for Koblo Studio and a bunch of plug-in's 
There is a roadmap and a suggestion list at koblo.com please contact max@koblo.com to get access.
There is additional documentation on http://koblo.com/wiki/

• Products: In this folder you find the code for Koblo Studio and the plug-in's
• SDK: The SDK contains common files shared among projects you need this to compile any of the products
• Tools: Here you find the code for a little resource compiler that builds a binary resource file to include to your project
